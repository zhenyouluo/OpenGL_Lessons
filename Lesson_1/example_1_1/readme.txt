1.1 Шаблон проекта. Интегрирование в qt opengl
Пример описывает простой шаблон OpenGL класса, созданного как QWidget.

ВНИМАНИЕ! В pro файле настроек Qt прописываем opengl.
Одна из его строк теперь должна выглядеть так:
QT       += core gui opengl

initializeGL() - настройки машины OGL

resizeGL(int w, int h) - метод вызывается при изменении размеров полотна OGL. Параметры текущую принимают высоту и ширину окна.

paintGL() - отвечает за прорисовку объектов на полотне
