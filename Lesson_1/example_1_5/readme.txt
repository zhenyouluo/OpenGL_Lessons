1.5. Установка библиотеки glut. Прорисовка с помощью библиотеки сферы.
Пример демонстрирует применение библиотеки glut. 

Также, ее установку, и приминение.

ДЛЯ WINDOWS СИСТЕМ:

1)ВНИМАНИЕ: В .pro файле необходимо, например, в самом конце, добавить или дополнить строчку: 
LIBS += glut32.lib

2)А ТАКЖЕ: необходимо поместить 3 файла(glut.h,glut32.lib,glut32.dll)
в папку где выполняется ваша программа(debug или realise).

3)В хедере, подключаем glut.h, и спокойно юзаем в скупе с qt)

ДЛЯ UNIX СИСТЕМ:(придется установить не glut, а freeglut который включает в себя glut)
1)В терминале устанавливаем пакеты:
sudo apt-get install freeglut3
sudo apt-get install freeglut3-dev 

2)Прописываем в .pro файл следующую строку: 
QMAKE_LIBS_OPENGL = -lGL -lglut

3)Инклюдим теперь существующую библиотеку #include <GL/freeglut.h>

4)И теперь, все таки, необходимо проинициализировать окно для glut'а:
в функции main пишем следующую строку:
glutInit(&argc, argv);
(понятно, что в main функции должны быть те же аргументы)
int main(int argc, char *argv[])
